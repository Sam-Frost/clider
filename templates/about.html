Q. What is Clider ?

Ans : Clider is a web application built as final project from Harvard's Introdcutory Course CS50.

Q. What does Clider do ?

Ans : Clider is a course builder. You can select any number lectures from YouTube and group them together into a course. Then you can track you progress using clider and upon completion of the course clider generates a certificates stating name of the person who completed the course, stating hours of lectures attended!

Technologies On Which Clider is built :

  • HTML5

    Clider is built on HTML and CSS but there is almost no use of javascript.

  • Python

    Clider backend is powered by python. It is running on the Flask micro web framework.

  • SQLite

    SQLIte3 is the choice of database for storing data regarding user login credentials, users coruses, their progress and their certificates.

  • Bootstrap

    Alot of components of this web application are built on top of template structures from bootstrap such as the navbars.
